Python

This is a basic intro to python, in this you will learn about; 
- varables 
- if statements 
- inputs 
- loops  
 
 
 
The first part will be variables, these are used in python to store data. 
There are several types of varibles in python: 
 
- string - a string stores text and numbers. You cannot use mathamatical operators such as * (multiply) on these. 
 
- int - a interger is a whole number, which means it cannot store decimal numbers. You can do mathamatical operators on these 
 
- float - a float is simlar to a int but it can store numbers with desimal places. 
 
- bool - a bool or boolean can only be True or False in python it, it is mainly used in loops. 
 
 
 
 
The second part is if statements, they will run if somthing is true, for exsample; 
 
x = 3 
if x == 3: 
    print("x = 3") 
 
this will output; 
 
x = 3 
 
 
 
 
 
The third part is inputs, inputs can be used to get an input for a user, for example; 
 
data = input("how are you") 
print("you wrote:") 
print(data) 
 
 
 
 
the forth part is loop, loops repeat the thing in them. there are 2 types of loops; while loops and range loops. for example; 
 
while True: 
    print("this will print forever") 
 
for i in range(100): 
    print("this will print 100 times")
